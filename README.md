# constructor-hangman-app
Create an hangman whit an prompt. The user will guess a letter for the word chosen by a randomizer from computation. User has ten chances to get the right answer. If the user guesses the chosen letter wrong ten time, game over.


start in letter.js
1 - create a letter function storing an empty string for the underscore and correct letter

start in word.js
2 - create an array for the word objects being guessed store wrong answer and console.log correct if the letters matches the object letters

game.js
3 - create an array for the objects you want the user to guess.

index.js
4 - download packages and append the letter chosen by user into the system
